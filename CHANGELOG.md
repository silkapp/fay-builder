# Changelog

## 0.2

* Add flag x-fay-strict-modules, specifying modules that should get a
  strict wrapper.
* Add flag x-fay-library, indicating that no main function should be
  called.
* Export new `readField` function.
* Signature for `fayConfig` has changed.

#### 0.1.0.7 (2014-04-29)

* Use `fay 0.20`

#### 0.1.0.5

* Initial release
